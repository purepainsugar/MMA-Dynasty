# WFC Lightweight Hand-Crafted Fighters

## Purpose

This document defines the first 12 hand-crafted fighters in the WFC Lightweight division.

These fighters are the identity anchors of the first playable division.

They should create:

- championship history
- contender drama
- prospect excitement
- veteran depth
- stylistic variety
- future rivalries
- memorable personalities

- Promotion: World Fighting Championship
